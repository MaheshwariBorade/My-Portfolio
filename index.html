Attendance Automation System Using AWS

The Attendance Automation System Using AWS project simplifies the process of tracking and recording student attendance using cloud technology. The system uses face recognition to identify students, making the attendance process more accurate and efficient. It relies on AWS services like AWS Lambda, Amazon S3, Amazon DynamoDB, and AWS Rekognition for cloud-based computing, secure data storage, and facial recognition. Attendance records are automatically updated in real time when students check in, reducing the need for manual tracking.

The Building a Serverless Application project focuses on creating an application that runs on cloud services without the need to manage any servers. Static Website Hosting on AWS S3

The Static Website Hosting on AWS S3 project involves hosting a simple, static website on Amazon S3 (Simple Storage Service),AWS Cognito,Authentication,API Gateway,Lambda,RDS. AWS S3 allows you to store and serve static files such as HTML, CSS, and JavaScript without the need for traditional web servers. In this project, you will upload your website files to an S3 bucket, configure the bucket for public access, and set up the necessary settings to serve the site over the internet.
